🚀 Quick Setup

Step 1: Install SendGrid Python library
pip3 install sendgrid
Step 2: Get SendGrid API Key
- Sign up at SendGrid.com
- Go to Settings → API Keys → Create
- Copy your key (starts with "SG.")
Step 3: Verify Your Sender
- Go to Sender Authentication
- Create new sender with your email
- Check email and verify
Step 4: Configure in Odoo
- Go to Settings → Technical → Outgoing Mail Servers
- Create new server
- Select "SendGrid API" as authentication
- Paste your API key
- Test connection
